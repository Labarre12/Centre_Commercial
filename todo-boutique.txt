boutique :

gestion de produit :
| Fonction             | Méthode HTTP | URL                                 | Description                                 | Filtrage / Options                               |
| -------------------- | ------------ | ----------------------------------- | ------------------------------------------- | ------------------------------------------------ |
| Ajouter un produit   | POST         | /api/boutiques/:idBoutique/produits | Ajouter un nouveau produit dans la boutique | ID catégorie, couleur, image, description        |
| Lister les produits  | GET          | /api/boutiques/:idBoutique/produits | Récupérer tous les produits de la boutique  | Filtrer par catégorie, couleur, promotion active |
| Voir un produit      | GET          | /api/produits/:idProduit            | Détails d’un produit                        | ID produit                                       |
| Modifier un produit  | PUT          | /api/produits/:idProduit            | Mettre à jour un produit existant           | Image, description, catégorie                    |
| Supprimer un produit | DELETE       | /api/produits/:idProduit            | Supprimer un produit                        | ID produit                                       |

profil :
| Fonction        | Méthode HTTP | URL                        | Description                                   | Filtrage / Options                             |
| --------------- | ------------ | -------------------------- | --------------------------------------------- | ---------------------------------------------- |
| Voir profil     | GET          | /api/boutiques/:idBoutique | Récupérer les informations de la boutique     | ID boutique                                    |
| Modifier profil | PUT          | /api/boutiques/:idBoutique | Mettre à jour les informations de la boutique | Libellé, description, horaires, URL, catégorie |

Suivi des clients :
| Fonction       | Méthode HTTP | URL                                    | Description                                            | Filtrage / Options                |
| -------------- | ------------ | -------------------------------------- | ------------------------------------------------------ | --------------------------------- |
| Liste clients  | GET          | /api/boutiques/:idBoutique/clients     | Voir tous les clients ayant acheté dans la boutique    | Filtrer par date, montant dépensé |
| Détails client | GET          | /api/clients/:idClient                 | Voir détails d’un client                               | Achats, historique, contact       |
| Clients VIP    | GET          | /api/boutiques/:idBoutique/clients/top | Voir clients qui achètent le plus ou dépensent le plus | Par période                       |

| ------------------ | ------------ | -------------------------------- | ----------------------------- | ------------------------------------- |

| Supprimer paiement | DELETE       | /api/loyer/:idLoyer              | Supprimer un paiement         | ID paiement                           |
