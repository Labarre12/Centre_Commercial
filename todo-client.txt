Authentification (login / inscription) :
| Fonction          | Méthode HTTP | URL                | Description                               | Filtrage / Options                                                             |
| ----------------- | ------------ | ------------------ | ----------------------------------------- | ------------------------------------------------------------------------------ |
| Inscription       | POST         | /api/auth/register | Créer un nouveau compte (client ou admin) | Nom, email, mot de passe, rôle (client/admin/propriétaire)                     |
| Connexion / Login | POST         | /api/auth/login    | Se connecter avec ses identifiants        | Email, mot de passe, retour token JWT                                          |
| Filtrage par rôle | POST         | /api/auth/login    | Login filtré selon le rôle                | Vérifie si le rôle correspond au compte (ex : client vs admin vs propriétaire) |
| Déconnexion       | POST         | /api/auth/logout   | Déconnecter le compte                     | Supprimer ou invalider le token côté serveur/client                            |
